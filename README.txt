#  Triticum aestivum  Annotation
This repository contains the annotation for the Triticum_aestivum_4.0 genome. 
The annotation files include a GTF file describing all genes and transcripts, both protein-coding and noncoding.

* ftp://ftp.ccb.jhu.edu/pub/data/Triticum_aestivum/Triticum_aestivum_4.0/
